# Chess Battle Royal firearm sound attributions

The GLTF firearm model sources used by Chess Battle Royal do not include source-authored audio tracks. To keep binary audio files out of this repository and pull request, the game streams the fallback firearm effects directly from their free Creative Commons Attribution 4.0 International source pages at runtime:

- Gunshot profile — "Gunshot Sound Effect" by Alexander / Orange Free Sounds, CC BY 4.0: https://orangefreesounds.com/gunshot-sound-effect/
  - Runtime file URL: https://www.orangefreesounds.com/wp-content/uploads/2015/01/Gunshot-sound-effect.mp3
- Shotgun profile — "Shotgun Sound" by Alexander / Orange Free Sounds, CC BY 4.0: https://orangefreesounds.com/shotgun-sound/
  - Runtime file URL: https://www.orangefreesounds.com/wp-content/uploads/2016/12/Shotgun-sound.mp3
- Launcher profile — "Cannon Sound Effect" by Alexander / Orange Free Sounds, CC BY 4.0: https://orangefreesounds.com/cannon-sound-effect/
  - Runtime file URL: https://www.orangefreesounds.com/wp-content/uploads/2017/03/Cannon-sound-effect.mp3
