# Bilardo Shqip (Rotation 61) – Product Rules

This mode is implemented as a **race to 61 points** using the numbered 1–15 ball set (solids and stripes visuals).  
Scoring uses each ball's number as points (e.g., potting 10 = 10 points).

## Core gameplay rules

1. Rack all 15 numbered balls in a triangle.
2. The cue ball must strike the **lowest-numbered ball still on the table** first.
3. Legal potted object balls score their face value.
4. Scratch (cue ball potted or off table) is a foul and gives ball-in-hand to opponent.
5. First player to reach **61 points** wins immediately.
6. If all object balls are gone before 61, higher score wins.

## Why this ruleset

In Albanian venues this format is commonly referred to as "61" / rotation-style play.  
Official Albanian federation pages list mainstream pool disciplines but do not publish a distinct, detailed "Bilardo Shqip 61" tournament appendix, so this implementation follows standard 15-ball rotation conventions adapted to the product requirement of race-to-61.
